Unavailable Times

Set the time ranges when this employee is not available

Field          Allowed values
-----          --------------
Minute         0-59
Hour           0-23
Day of month   1-31
Month          1-12
Day of week    0-7
Year           1970-2100

To set the time periods in which an employee is not available, we write the time period in a special format. This format defines the time in which the employee should not work for each individual unit of time. We start from the smallest unit of time and continue to the largest. You can view the units of time in the example table to the left.

You are allowed to use commas(representing a list), dashes(representing a range), a star(meaning any value), or just a single number to convey what times an employee is not available.

As an example, let's go through how you would write an employee is unavailable from 12/25/2020 to 12/30/2020.

  1. The first time period we define is minute. Since for this example we don't want the employee to be unavailable for entire days, we can put a *. This signifies that, for this time period, all minutes are unschedulable.
  2. The second is the hour time period. It follows that we also don't want to specify hours. So we put another *.
  3. The third is day of month or date. For this we could choose to use a list like so: 25,26,27,28,29,30. But thats a lot of writing. Instead, we use a range: 25-30.
  4. The third is Day of month. For this we could use a list like so: 25,26,27,28,29,30. But thats a lot of writing. Instead we use a range: 25-30.
  5. The fourth is Days of the week. For this we don't mind which days the 25-30 fall on. So as usual we leave a *.
  6. Lastly, we need to define the year or else this employee gets Christmas off every year. So we just use a single value: 2020.

Altogether the full expression becomes: * * 25-30 12 * 2020
